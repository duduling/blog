## Solution

#### My Solution

```js
function solution(s, n) {
    return s.split('').map(item => {
        if(item === ' ') {
            return item
        }
        
        const boundary = item < 'a' ? 65 : 97
        
        return String.fromCharCode(((item.charCodeAt() - boundary + n) % 26) + boundary)
    }).join('')
}
```
