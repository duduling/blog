# 86419. 최소직사각형

## Solution

```js
function solution(sizes) {
  const longList = [];
  const shortList = [];

  sizes.forEach(([w, h]) => {
    if (0 < w - h) {
      longList.push(w);
      shortList.push(h);
    } else {
      longList.push(h);
      shortList.push(w);
    }
  });

  return Math.max(...longList) * Math.max(...shortList);
}
```

## Review

사각형의 큰 값과 작은 값을 따로 분리해 큰 값의 최댓값 작은 값의 최댓값을 곱해준다.
