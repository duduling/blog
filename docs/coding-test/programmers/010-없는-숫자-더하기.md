# 없는 숫자 더하기 (Level 1)

## Solution

### My Solution

```js
function solution(numbers) {
  const TOTAL_NUM = 45;

  return TOTAL_NUM - numbers.reduce((acc, num) => (acc += num), 0);
}
```

## Review

Total Num을 구하는 것을 동적으로 만들지 그냥 하드 코딩할지 고민하다가 그냥 하드코딩으로 했다. (조건에 따라 때로는 하드 코딩이 더 좋은 듯?)
