# 폰켓몬 (Level 1)

## Solution

### My Solution

```js
function solution(nums) {
  const totalPokemonCount = new Set(nums).size;
  const limitChoice = Math.floor(nums.length / 2);

  return limitChoice > totalPokemonCount ? totalPokemonCount : limitChoice;
}
```
