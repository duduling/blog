# 최소직사각형 (Level 1)

## Solution

### My Solution

```js
function solution(sizes) {
  const answer = sizes
    .map(([a, b]) => (a > b ? [a, b] : [b, a]))
    .reduce(
      (acc, cur) => {
        return [Math.max(acc[0], cur[0]), Math.max(acc[1], cur[1])];
      },
      [0, 0]
    );

  return answer[0] * answer[1];
}
```
