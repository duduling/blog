# 약수의 개수와 덧셈

## Solution

```js
function solution(left, right) {
  let answer = 0;

  for (let i = left; i <= right; i++) {
    if (Number.isInteger(Math.sqrt(i))) {
      answer -= i;
    } else {
      answer += i;
    }
  }

  return answer;
}
```

## Review

숫자를 제곱근 했을 때 정수면 약수가 홀수이다.
