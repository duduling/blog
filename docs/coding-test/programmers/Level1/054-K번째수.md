# K번째수

## Solution

```js
function solution(array, commands) {
  return commands.reduce(
    (acc, [start, end, target]) => [
      ...acc,
      array.slice(start - 1, end).sort((a, b) => a - b)[target - 1],
    ],
    []
  );
}
```
