# 콜라츠 추측

## Solution

```js
function solution(num, cnt = 0) {
  if (num === 1) return 500 <= cnt ? -1 : cnt;

  return solution(num % 2 === 0 ? num / 2 : num * 3 + 1, ++cnt);
}
```
